個人のポートフォリオ・ブログサイトです。Works、Blog、読書記録、登壇情報などをまとめています。

## 技術スタック

- **Astro**: コンテンツファーストなスタティックサイトジェネレータ
- **Tailwind CSS v4**: ユーティリティファースト CSS
- **TypeScript**: 型安全なコンテンツ管理
- **Cloudflare Workers**: エッジ配信
- **Biome**: リンター・フォーマッター統合
- **Graphite**: スタック型 PR ワークフロー

## 特徴

- **コンテンツコレクション**: Zod スキーマで型安全な Markdown / MDX 管理
- **OGP リンクカード**: URL だけでリッチなリンクカードを表示
- **RSS 統合**: Zenn / Note / Podcast の外部記事を自動取得
- **自動 OG 画像生成**: ブログ記事ごとに動的に OG 画像を生成
- **Claude Code Review**: PR 作成時に AI が自動レビュー
