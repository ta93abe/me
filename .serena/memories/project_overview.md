# プロジェクト概要

## 目的
個人ポートフォリオ/ランディングページプロジェクト。

## 説明
モダンなスタティックサイトジェネレータ(Astro)を使用した個人ウェブサイト。
高速なページロード、優れた開発体験、そしてCloudflareエッジネットワークでのグローバル配信を実現。

## 主要機能
- ブログ記事の管理
- プロジェクト作品の展示
- 読書記録（bookshelf）
- 登壇情報（talks）
- Qiita RSS統合
- サイトマップ生成
- SEO最適化（Open Graph、Twitter Card、構造化データ）
- ページファインド（検索機能）

## デプロイ
Cloudflare Workersにデプロイされ、https://ta93abe.com でホスティング。

## プロジェクトタイプ
静的サイトジェネレーター (SSG) ベースのポートフォリオサイト
