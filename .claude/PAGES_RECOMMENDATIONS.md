## 追加推奨ページ

### 優先度：高（必須）

#### 1. Contact（お問い合わせ）
- **URL**: `/contact`
- **目的**: 訪問者からの連絡を受け付ける
- **実装内容**:
  - コンタクトフォーム（名前、メール、件名、メッセージ）
  - メールアドレス・SNSリンクの表示
  - フォーム送信（Cloudflare Workers対応）またはmailto:リンク
- **重要度**: ★★★★★
- **理由**: ビジネス機会獲得に不可欠

#### 2. Skills（スキル・専門分野）
- **URL**: `/skills`
- **目的**: 技術スタック・専門性の可視化
- **実装内容**:
  - プログラミング言語（レベル表示）
  - フレームワーク・ライブラリ
  - ツール・プラットフォーム
  - ソフトスキル
  - 視覚的な表現（アイコン、プログレスバー）
- **重要度**: ★★★★★
- **理由**: 採用担当者・クライアントが最も知りたい情報

---

### 優先度：中（推奨）

#### 3. Experience / Career（職務経歴）
- **URL**: `/experience`
- **目的**: キャリアの詳細な説明
- **実装内容**:
  - タイムライン形式の職務経歴
  - 各ポジションでの成果・実績
  - 使用技術
  - 期間（開始日〜終了日）
- **重要度**: ★★★★☆
- **理由**: プロフェッショナルな印象を与える
- **備考**: Aboutページに統合する選択肢もあり

#### 4. Resume / CV（履歴書ダウンロード）
- **URL**: `/resume`
- **目的**: 履歴書のダウンロード提供
- **実装内容**:
  - PDFダウンロードリンク
  - オンライン版レジュメの表示
  - 印刷最適化版
- **重要度**: ★★★★☆
- **理由**: 採用プロセスで頻繁に要求される

#### 5. Services（提供サービス）
- **URL**: `/services`
- **目的**: フリーランス・コンサルティング提供サービスの説明
- **実装内容**:
  - サービス内容の詳細
  - 料金体系（オプション）
  - 提供可能な範囲
- **重要度**: ★★★☆☆
- **理由**: フリーランス活動する場合に必要
- **備考**: フリーランスでない場合は不要

---

### 優先度：低（オプション）

#### 6. Testimonials / Recommendations（推薦・評価）
- **URL**: `/testimonials`
- **目的**: クライアント・同僚からの評価を表示
- **実装内容**:
  - 推薦文
  - 推薦者の情報（名前、役職、会社）
  - 星評価・スコア
- **重要度**: ★★★☆☆
- **理由**: 信頼性向上に効果的
- **備考**: 十分な推薦文が集まってから実装

#### 7. Awards / Certifications（受賞歴・資格）
- **URL**: `/awards` または `/certifications`
- **目的**: 資格・受賞歴の証明
- **実装内容**:
  - 取得資格一覧
  - ハッカソン・コンテスト受賞歴
  - 認定証のバッジ
- **重要度**: ★★☆☆☆
- **理由**: 専門性の証明になる
- **備考**: Skillsページに統合する選択肢もあり

#### 8. Uses（使用ツール・機材）
- **URL**: `/uses`
- **目的**: 開発環境・愛用ツールの紹介
- **実装内容**:
  - エディタ・IDE設定
  - ハードウェア（PC、キーボード等）
  - ソフトウェア・アプリ
  - デスクセットアップ
- **重要度**: ★★☆☆☆
- **理由**: 開発者コミュニティで人気のコンテンツ
- **備考**: ブログ記事としても可

#### 9. Now（現在の活動）
- **URL**: `/now`
- **目的**: 現在取り組んでいること・学習中の技術
- **実装内容**:
  - 現在のプロジェクト
  - 学習中の技術
  - 読んでいる本
  - 最終更新日
- **重要度**: ★★☆☆☆
- **理由**: パーソナルブランディングに効果的
- **備考**: 定期的な更新が必要

### 法的・ポリシー系（状況に応じて）

#### 10. Privacy Policy（プライバシーポリシー）
- **URL**: `/privacy`
- **重要度**: ★★★☆☆

- **URL**: `/terms`
- **重要度**: ★☆☆☆☆

## 実装優先順位の提案

### フェーズ1: 必須ページ（すぐ実装）
1. **Contact** - お問い合わせフォーム
2. **Skills** - 技術スタック可視化

### フェーズ2: 推奨ページ（中期）
3. **Experience** - 職務経歴（またはAbout拡充）
4. **Resume** - 履歴書ダウンロード

### フェーズ3: オプション（状況に応じて）
5. **Services** - フリーランス活動する場合
6. **Testimonials** - 推薦文が集まったら
7. **Now** - パーソナルブランディング強化時

## 既存ページの改善提案

### 1. ホームページ（`/`）の強化
現在はWelcomeコンポーネントのみ。以下を追加推奨：
- ヒーローセクション（名前、キャッチコピー、CTA）
- 最新のWorks（3件）
- 最新のBlog記事（3件）
- スキルハイライト
- CTAボタン（Contact、Works、Resume）

### 2. Aboutページの拡充
- 経歴サマリー
- 得意分野
- 興味・関心
- 写真・画像

### 3. ナビゲーションの整理
現在6項目。Contactを追加すると7項目になり、多くなる可能性。
- 主要ページのみヘッダーに配置
- その他はフッターまたはドロップダウンメニュー

## 技術的考慮事項

### Contactフォーム実装
- Cloudflare Workers + Email Workers
- または外部サービス（Formspree、Netlify Forms等）
- バリデーション
- スパム対策（reCAPTCHA）

### コンテンツコレクション追加
以下のコレクション追加を検討：
- `skills` - スキル情報
- `experience` - 職務経歴
- `certifications` - 資格情報

### ナビゲーション改善
- モバイル対応ハンバーガーメニュー
- アクティブリンクのハイライト
- ドロップダウンメニュー（オプション）
